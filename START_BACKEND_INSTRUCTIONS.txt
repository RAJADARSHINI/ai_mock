STEP 1: START THE BACKEND
================================================================================

Choose ONE backend to run:

PYTHON BACKEND (Flask):
  1. Open Terminal in VS Code
  2. Run: cd python_backend
  3. Run: python app.py
  4. Keep terminal open - you'll see: http://localhost:5000

NODE BACKEND (Express):
  1. Open Terminal in VS Code
  2. Run: cd backend
  3. Run: npm install (first time only)
  4. Run: npm start
  5. Keep terminal open - you'll see: http://localhost:5000

================================================================================
STEP 2: START THE FRONTEND SERVER (if needed)
================================================================================

If your frontend files need a server:

  1. Open NEW Terminal in VS Code
  2. Run: python -m http.server 8000
  3. Keep this terminal open too

================================================================================
STEP 3: OPEN IN EXTERNAL BROWSER
================================================================================

  1. Open Chrome or Edge (external application, NOT in VS Code)
  2. Navigate to: http://localhost:8000
  3. **Grant camera and microphone permissions when prompted**
  4. Your app should now work with full camera/mic access!
